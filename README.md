# Kasper Template

A responsive static website template built with **HTML** and **CSS**.

## Overview

This project is a frontend template called **Kasper**.  
It includes a modern multi-section landing page layout with common website sections such as:

- Home / Landing
- Services
- Portfolio
- About
- Pricing
- Contact

This repository is useful as a UI practice project and as a base for future frontend work.

## Tech Stack

- HTML
- CSS
- Font Awesome
- Google Fonts

How to Run

Clone the repository

Open index.html in your browser

Purpose

This project was built to practice:

Layout structure

Responsive design

Section-based landing pages

Styling with external assets and fonts
